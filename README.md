# Gestion Restaurants

Bienvenue dans le projet **Gestion Restaurants** !

## Description
Cette application utilise une architecture microservices pour offrir une plateforme de gestion et de réservation de restaurants. Les utilisateurs peuvent:
- Rechercher des restaurants par emplacement et cuisine
- Lire et laisser des avis
- Réserver des tables en ligne

## Technologies Utilisées
- Node.js pour le backend
- MongoDB pour la base de données
- React pour le frontend
- Docker pour la conteneurisation des microservices

## Fonctionnalités Principales
1. Microservice d'authentification
2. Microservice de gestion des restaurants
3. Microservice de gestion des réservations
4. Microservice d'évaluation des restaurants

## Installation
Clone ce dépôt et exécutez:
```bash
npm install
npm start
```
